Easily Build Responsive Sites

A fully responsive grid system and convenient .classes for content control.
Use for any project, build sites quickly and efficiently.


Download

Features

Scalable

Built using percentage values, your content fluidly fits all screen sizes

Easy Syntax

Easy to read & remember .class names using simple words (i.e. "full-width-s, fourth-m")

Content-Control

Control what you want visitors to see with convenient classes such as .hidden, .only, and more!.

Build Mockup Layouts

Put together a responsive prototype for a client quickly. No need to copy and paste dummy text, use .lorem-text instead.

Perfect Fit

Plugs right into your HTML and work alongside any other CSS files.

No Hacking Needed

Minimal and lightweight. No time wasted searching and overriding predefined styles.

About PEARcss

Inspired by Twitter's BootStrap, and the many new web developers learning CSS, I decided to make this grid system for ease of use for myself and for others to build sites quickly and have fun.
